## Each language follows its ecosystem standard ##

# standards

- generic standards for ur-* applications are all sitting in ../!ur-code-template/!coding-standards/,
- generic standards for ur-* applications are indexed and summarized in file ../!ur-code-template/!coding-standards/!generic-coding-standards.md,
- ../!ur-code-template/coding-standards/!generic-coding-standards.md points to other md files, each clarifying standards on a specific angle,
- ../!ur-code-template/!coding-standards/generic-coding-standards.md points to other md files path relative to itself's path,
- specific standards to one ur-* application are all sitting in ./!coding-standards in a similar pattern to generic standard:
- specific standards to one ur-* application are indexed and summarized in file ./!coding-standards/!specific-coding-standards.md,
- ./!coding-standards/!specific-coding-standards.md points to other md files path relative to itself's path,
- specific standards can override generic standards.
